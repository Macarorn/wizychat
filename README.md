# Project Setup

**Prerequisites:**
````markdown
- Node.js (>= 14.x)
- npm (>= 6.x)
````


## Installation

1. Clone the repository:
````markdown
   ```bash
   git clone <repo-url>
   cd <repo-directory>
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run start
   ```

4. Open your browser and visit:

   ```
   http://localhost:3000
   ```
